## 介绍

WebService 是对 HTTP 协议的二次封装，可以理解为 HTTP + XML



## wsdl 文件

wsdl 文件是服务端提供的 webservice 接口说明书（xml 格式）

根据 wsdl 文件，可以生成 Java 文件，也可以直接发起 http 请求，请求体是 xml
